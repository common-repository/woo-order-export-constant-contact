== Description ==
Woo order export to constant contact plugin export users data from woocommerce order and merge into constant contact.

In general, this plugin is a Constant Contact plugin for WooCommerce. 

Woo order export to constant contact plugin export woocommerce users data from woocommerce order and merge into constant contact.

This is the very first plugin on WordPress by which store administrator can send the WooCommerce order data directly to Constant Contact account list without any hassle.

Woo order export to constant contact plugin helps any WooCommerce store to collect data as well as export data into Constant Contact to run any promotion campaign.

Woo order export to constant contact plugin export woocommerce customer order email address, first Name and last name from woocommerce order and send it to Constant Contact account.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly by seraching "Woo Order Export To Constant Contact" .
2. Activate the plugin through the 'Plugins' screen in WordPress
3. WooCommerce -> Settings -> Constant Contact
